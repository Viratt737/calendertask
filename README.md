# Wall Calendar Web App

A modern and interactive wall-style calendar built using React and Tailwind CSS. This project allows users to select date ranges and attach notes, providing a clean and intuitive user experience.

---

## Features

* Interactive monthly calendar view
* Select single date or date range
* Add and save notes for selected dates
* Hover-based range preview
* Highlight current day and weekends
* Responsive UI design
* Smooth and minimal user interface

---

## Tech Stack

* React (Functional Components and Hooks)
* Vite (Fast build tool)
* Tailwind CSS (Utility-first styling)

## Installation and Setup

1. Clone the repository

```
git clone https://github.com/your-username/wall-calendar.git
cd wall-calendar
```

2. Install dependencies

```
npm install
```

3. Run the development server

```
npm run dev
```

4. Open in browser

```
http://localhost:5173
```

---

## Usage

* Click on any date to start selecting a range
* Click another date to complete the range
* Add a note for the selected range
* Saved notes will appear below the calendar
* Navigate between months using arrow buttons
